## Getting started

### Prerequisites

- Node.js (v14 or newer) and npm installed  
- MongoDB (local or hosted) if the backend requires it

### Install & run backend

```bash
# from repository root
cd backend

# install dependencies
npm install

# create .env file (see Environment variables below)
# start server
npm run start
# or, for development (if available)
npm run dev
